# Automatic-CLOUD-and-SHADOW-mask-generation-from-Resourcesat-2-2A-Liss4-Satellite-Images
Automated cloud, shadow and no-cloud masking framework for Resourcesat-2/2A LISS-IV imagery. A pixel-wise Multi-Layer Perceptron (MLP) classifier is trained on TOA reflectance derived from Green, Red and NIR bands using open-source Python libraries.
